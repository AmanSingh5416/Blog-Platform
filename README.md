# 📝 Blogging Platform

A full-stack blogging web application built using the MERN stack that allows users to create, edit, and manage blogs with authentication and a clean UI.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* ✍️ Create, Edit, Delete Blogs
* ❤️ Like & Comment System
* 📱 Responsive Design
* ⚡ Fast and optimized performance

---

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML5, CSS3, JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose ODM)

### Tools & Packages

* dotenv
* bcrypt (Password hashing)
